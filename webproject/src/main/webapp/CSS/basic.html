CSS(Cascading Style Sheet)

HTML문서의 요소(tag)들 중에서 선택적으로 스타일(서식)을 적용할 수 있는 기능이다.

사용자 스타일 지정 형식 3가지 : 1.인라인 스타일 형식, 2.내부 스타일 형식, 3.외부 스타일 형식

1.인라인 스타일 형식 : 요소(tag)내에 style 속성으로 스타일(서식)을 지정하는 방법

레드향

껍질에 붉은 빛이 돌아 레드향이라 불린다.

레드향은 한라봉과 귤을 교배한 것으로 일반 귤보다 2~3배 크고 과육이 붉고 통통하다.

비타민 C와 비타민 P가 풍부해 혈액순환, 감기예방등에 좋은 것으로 알려져 있다.

2.내부 스타일 형식 : 적용할 스타일들을 문서 내부에 한 곳에 모아놓고 스타일을 적용하는 방법

head 영역내에 style 태그로 지정한다.

형식)선택자(selector){ 속성1 : 속성값1; 속성2 : 속성값2; }
