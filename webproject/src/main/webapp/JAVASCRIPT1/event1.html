이벤트(event)와 이벤트 처리기(eventListener / autoHadler)

이벤트(event): 웹문서내의 행요소(태그)에서 이루어지는 사건(동작이다.)

이벤트 처리기(eventListener/eventHandler) : 이벤트가 발생했을때 처리 기술(실행할 함수)이다.

이벤트 처리기 방식 : 1)인라인 방식, 2)프로퍼티 방식, 3)addEventListener 방식

1)인라인(inline)방식 : 이벤트(사건,동작)를 HTML 요소(태그)의 속성으로 직접 지정하는 방식이다.

형식) <태그 on이벤트명 = "함수명">
