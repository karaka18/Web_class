input tag 주요 속성

1.autocomplete 속성 : 폼이나 폼 요소에 자동완성 기능의 지원 여부를 지정(checkbox, radio, 버튼 유형 제외) 형식)autocomplete = "on/off" 2.autofocus 속성 : 페이지가 로드 될 때 특정 폼 요소에 자동으로 커서가 표시(get focus)되도록 지정 3.placeholder 속성 : 폼 요소에 어떤 내용을 입력해야 하는지 간단한 설명(힌트) 표시 형식)placeholder ="설명(힌트)" 4.readonly 속성 : 사용자가 입력하지는 못하고 읽기만 할 수 있는 읽기 전용 폼 요소로 만듬 5.required 속성 : 필수 입력 필드를 지정(지정한 폼 요소값에 데이터가 반드시 입력되어야만 서버로 전송할 수 있음)
