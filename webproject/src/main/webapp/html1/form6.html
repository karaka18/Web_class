11)textarea 태그 : 여러줄을 입력하는 텍스트 영역을 만듬

style="resize:none;" 속성으로 크기 조절 못하도록 지정

12)select, option 태그 : 드롭다운목록(drop down list)을 만듬

value속성 : 서버에 넘겨주는 값을 지정해주어야 함!!!

selected 속성 : 기본적으로 선택되어 있는 값을 지정

select 태그의 속성 지정

size속성 : 화면에 표시할 목록 개수, multiple 속성 : 2개 이상의 항목을 동시에 선택 가능 (Ctrl + 클릭: 따로 선택, Shift + 클릭: 연속해서 선택)
