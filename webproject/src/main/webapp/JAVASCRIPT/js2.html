출력하기 : 1.알림창 : alert(), 2.콘솔창: console.log(), 3.HTML 문서에 출력:document.wirte()

1.알림창 : alert() -> 문서에는 표시되지 않고 알림창 형식으로 브라우저 위쪽에 확인단추로 표시된다.

HTML 태그를 넣어서 사용할 수 없다.

2.콘솔창: console.log() -> 브라우저내에서 소스코드의 오류를 발견하거나 변수값을 확인할 수 있는 창에 표시

크롬 브라우저에서 콘솔창 확인 : Ctrl + Shift + J or F12

HTML 태그를 넣어서 사용할 수 없다.

3. HTML 문서에 출력:document.wirte() -> body 영역에 표시된다.

HTML 태그를 넣어서 사용할 수 있다.
